# My Cabinet – Medical Management System

Java desktop application for managing a medical cabinet.

## Features
- Patient management
- Appointment scheduling
- Medical records storage

## Technologies
- Java
- Java Swing (or JavaFX if used)

## How to run
Compile the project:
javac -d bin src/**/*.java

Run:
java -cp bin Main

## Prerequisites

To compile and run this project from source, you need:

- Java JDK installed
- JavaFX installed (if using JavaFX)
- MySQL Connector library for database connection
- PDFBox library for PDF handling

**Note:**  
This repository contains only the source code. Precompiled JRE, libraries, and drivers are not included. Please install the required dependencies before running the project.
