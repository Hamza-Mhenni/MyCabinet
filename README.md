# My Cabinet – Medical Management System

Java desktop application for managing a medical cabinet.

## Features
- Patient management
- Appointment scheduling
- Medical records storage

## Technologies
- Java
- Java Swing (or JavaFX if used)

## How to run
Compile the project:
javac -d bin src/**/*.java

Run:
java -cp bin Main
